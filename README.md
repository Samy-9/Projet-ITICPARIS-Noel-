Concours de Pull de Noël – Application type Tinder
Ce projet est une application web permettant d’organiser un concours de pulls moches de Noël avec une interface de vote inspirée de Tinder (swipe droite pour liker, swipe gauche pour disliker).
L’objectif est de proposer une expérience simple : chaque utilisateur s’inscrit, dépose la photo de son pull, puis vote pour les pulls des autres participants.

1. Fonctionnalités principales
Inscription / Connexion

Création de compte avec nom, email Yparéo et mot de passe.

Connexion à partir des comptes existants.

Vérification simple de l’unicité des emails.

Gestion du profil

Affichage du nom et de l’email de l’utilisateur connecté.

Upload d’une photo de pull.

Recadrage automatique de l’image en carré 300×300 via un canvas.

Sauvegarde de la photo dans le stockage local du navigateur.

Système de vote type Tinder

Affichage d’une carte par participant (photo + nom + compteurs de likes/dislikes).

Swipe à droite pour “like”, swipe à gauche pour “dislike”.

Boutons ❤️ / ❌ pour voter sans swipe.

Barre de progression indiquant le nombre de pulls déjà votés.

Écran de fin lorsque tous les participants ont été notés.

Données et persistance

Utilisation du localStorage pour stocker :

les utilisateurs (étudiants) et leurs informations,

les votes effectués,

l’utilisateur actuellement connecté.

Jeu de données de démo (utilisateurs test avec photos de pulls).
